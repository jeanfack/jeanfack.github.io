Comment configurer une source de donnée SQLite sur Open Office Calc ?

  1. Lancer Open Office Base.



  2. Dans la fenêtre Assistant Base de données :
    1. Etape 1: Sélectionner une base de donnée
      1. Selectionner "Se connecter à une base existante"
      2. Selectionner "ODBC"
      3. Cliquer sur "Suivant"



    2. Etape 2: Paramétrer une connexion ODBC
      1. Cliquer sur "Parcourir".



      2. Dans la fenêtre Source de données :
      3. Selectionner "ex1_sqlite3_32bits_sync_NORMAL" (Idem pour "ex1_sqlite3_64bits_sync_NORMAL").
      4. Cliquer sur "OK".



      5. Cliquer sur "Suivant".



    3. Etape 3: Paramétrer l'authentification de l'utilisateur



      1. Cliquer sur "Tester la connexion".
      2. Dans la fenêe "Test de connexion" :
        1. Vérifier que la connexion est établie.
        2. Cliquer "OK".



      3. Cliquer sur "Suivant".



    4. Etape 4: Enregistrer et continuer



      1. Selectionner "Oui, je souhaite que l'assistant référence la base de données".
      2. Cocher "Ouvrir la base de données pour édition".
      3. Cliquer sur "Terminer".



    5. Dans le fenêtre Enregistrer sous
      1. Parcourir: "C:\sqlite_odbc_sample\scripts\open_office\".
      2. Saisir dans nom de fichier: "ex1_sqlite3_32bits_sync_NORMAL.odb".
      3. Cliquer "Enregistrer".



  3. Dans la fenêtre "ex1_sqlite3_32bits_sync_NORMAL.odb - OpenOffice Base" :



    1. Selectionner "Tables" dans la zone "Base de données".
    2. Double cliquer sur la table "USER".



  4. Comment créer dans Calc un tableau croisé à partir de la base de donnée ?
    1. Lancer Open Office Calc.
    2. Créer un nouveau classeur.



    3. Enregistrer ce classeur sous C:\sqlite_odbc_sample\scripts\open_office\ex1.ods.
    4. Selectionner la feuille "Sheet1".
    5. Aller dans Données->Tableau croisé->Créer.



    6. Dans la fenêtre "Selectionner la source" :



      1. Selectionner "Source de données enregistrée sous OpenOffice".
      2. Cliquer sur "OK".



    7. Dans la fenêtre Tableau croisé :



      1. Deplacer les champs 'ID' et 'NAME' dans "Champs de ligne".
      2. Cliquer sur "OK".



    8. Afficher la feuille "Feuille1".



  5. Comment actualiser le tableau croisé ?
    1. Lancer cmd.exe dans C:\sqlite_odbc_sample\scripts.
    2. Executer ex1.INSERT.Charlie.Chaplin.cmd.



    3. Actualiser le tableau croisé.



    4. Observer l'insertion du nouvelle enregistrement.



    5. Executer ex1.DELETE.Charlie.Chaplin.cmd.



    6. Actualiser et observer la suppression de l'enregistrement.
